# App Logo Assets

Please place the KeepJoy app logo images in this directory with the following names:

- `app_logo.png` - The main app logo (white feather icon)
- `app_logo_with_bg.png` - Logo with gradient background (optional)
- `app_logo_outline.png` - Outline version (optional)

## Logo Details from User:
The user provided three versions of the logo:
1. Full app logo with purple-to-mint green gradient background and white feather icon
2. White feather icon on transparent background
3. Outlined feather icon on transparent background

The logo is a stylized white feather icon that represents lightness and joy.

## Usage:
- Welcome page: Uses app_logo.png (centered, white background container)
- Login page: Uses app_logo.png (centered, white background container)
- Profile page: Can use for account avatar or header

The gradient colors used in the app match the logo:
- Purple: #6B5CE7
- Mint Green: #5ECFB8
